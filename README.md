# 🧮 Умный Калькулятор

Многофункциональное веб-приложение с поддержкой PWA (установка на телефон как приложение).

## ✨ Возможности

### 🧮 Калькулятор (3 режима)
- **Обычный**: +, −, ×, ÷, √, x², %, |x|, n!
- **Научный**: 30+ функций (sin, cos, tan, log, ln, π, e, xʸ, ³√, asin, acos, atan, eˣ, 1/x и др.)
- **Программистский**: 4 системы счисления (DEC, BIN, OCT, HEX) + побитовые операции (AND, OR, NOT, XOR, <<, >>)

### 🔄 Конвертер единиц (7 типов)
- 📏 Длина (метры, км, мили, ярды, футы, дюймы)
- ⚖️ Вес (кг, г, фунты, унции, тонны)
- ⏱️ Время (сек, мин, часы, дни, недели, месяцы, годы)
- 🌡️ Температура (°C, °F, K)
- 🚀 Скорость (м/с, км/ч, mph, узлы)
- 📐 Площадь (м², км², гектары, акры, фут², дюйм²)
- 🥛 Объём (литры, мл, галлоны, кварты, пинты, м³, см³)

### ⏰ Таймер событий
- Расчёт разницы между двумя датами
- Поддержка имени события
- Точность до секунд

### 💱 Конвертер валют
- 20 валют с реальными курсами (open.er-api.com)
- График с 30 точками данных
- 4 временных периода: минуты, часы, дни, месяцы
- Подсветка курсов при наведении

### ⚙️ Настройки
- 🌐 **Язык**: Русский / English (автоопределение по браузеру)
- 🎨 **Тема**: ☀️ Светлая / 🌙 Тёмная / 🔄 Авто (по системе)
- 💝 Донат на разработку
- Сохранение настроек в localStorage

## 🚀 Запуск

### Предварительные требования
- Node.js 16+
- npm или yarn

### Установка
```bash
cd calculator-app
npm install
npm start
```

Приложение запустится на http://localhost:3000

## 📱 Установка как мобильного приложения (PWA)

Подробная инструкция в файле [PWA_INSTALL.md](PWA_INSTALL.md)

### Быстрый старт:
1. Откройте http://localhost:3000
2. Нажмите три точки (Chrome) или кнопку "Поделиться" (Safari)
3. Выберите "Установить приложение" / "На экран 'Домой'"
4. Готово! Приложение появится на рабочем столе

## 📦 Технологии

- **React 19** - современный UI-фреймворк
- **TypeScript** - типобезопасность
- **CSS3** - адаптивный дизайн и анимации
- **PWA** - Progressive Web App (установка на телефон)
- **localStorage** - сохранение настроек и истории
