# 🧮 Умный Калькулятор

Многофункциональный калькулятор с **Tauri v2** сборкой для Android: калькулятор, конвертер, таймер, валюта и настройки.

## ✨ Возможности

### 🧮 Калькулятор
- **Обычный режим**: +, −, ×, ÷, √, x², π, e, n!, mod (остаток от деления)
- **Научный режим**: sin, cos, tan, ctg, log, ln, asin, acos, atan, atg, π, e, x!, |x|, 1/x, % от, 10ˣ
- **Программистский режим**: DEC, BIN, OCT, HEX + AND, OR, XOR, NOT, <<, >>

### 🔄 Конвертер
Конвертер единиц: длина, вес, время, температура, скорость, площадь, объём.

### ⏰ Таймер
Отсчёт до особых дат: Новый год, Рождество (католическое/православное), Хэллоуин.

### 💱 Валюта
15 валют с офлайн-курсами и кнопкой обмена.

### ⚙️ Настройки
- **Язык**: English / Русский (авто-определение)
- **Тема**: ☀️ Светлая / 🌙 Тёмная / 🔄 Авто (настройки системы)
- **Скачать Android**: Только для веб-версии (скрыто в приложении Tauri)

## 🚀 Быстрый старт

```bash
cd calculator-app
npm install
npm start
```

## 📦 Сборка для продакшена

```bash
npm run build
```

## 🌐 Деплой на GitHub Pages

```bash
npm run deploy
```

## 🤖 Сборка Android APK

```bash
bash build-apk.sh
```

Результат: `apk/SmartCalc.apk` (~32 МБ)

### Требования
- Android SDK с NDK 25.2.9519653
- Java Development Kit (JDK)
- Keystore в `apk/keystore.jks` (пароль: `smartcalc123`)

## 📱 Технологический стек

- **Frontend**: React 18 + TypeScript
- **Mobile**: Tauri v2 (Android)
- **Стилизация**: Кастомный CSS (без фреймворков)
- **Сборка**: Create React App

```
calculator-app/
```

## 🎨 Особенности UI

- Адаптивный дизайн (mobile-first)
- Плавные анимации (кнопка обмена, переходы)
- Кнопки оптимизированы для тача
- Убран синий эффект нажатия на Android
- Поддержка тёмной/светлой темы
- Локальная история (последние 10 вычислений)

## 🧮 Логика калькулятора

| Режим | Особенности |
|-------|-------------|
| Обычный | π и e вставляются как символы (вычисляются на =), mod = остаток |
| Научный | % от = процент от числа, 10ˣ, тригонометрия в градусах |
| Программистский | Побитовые операторы бинарные (AND, OR, XOR), NOT унарный, подсветка базы |
