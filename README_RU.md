# 🧮 Умный калькулятор

Многофункциональный калькулятор с поддержкой сайта и сборки **Tauri v2** для Android, включающий
калькулятор, конвертер, таймер, обмен валют и настройки.

Сайт: https://larfi44.github.io/smart-calc/

## ✨ Функции

### 🧮 Калькулятор

- **Обычный режим**: +, −, ×, ÷, mod (остаток), √, x², π, e, rnd (округление),
- **Научный режим**: sin, cos, tan, ctg, log, ln, asin, acos, atan, atg, π, e, x!, |x|, 1/x, % от,
  факториал, модуль и так далее
- **Режим программиста**: +, -, ×, ÷, mod (остаток), системы учета (DEC, BIN, OCT, HEX), двоичные
  операции (AND, OR, XOR, NOT, <<, >>)

### 🔄 Конвертер

Конвертер единиц измерения для: длины, веса, времени, температуры, скорости, площади, объема.

### ⏰ Таймер

Обратный отсчет до важных дат: Новый год, Рождество (католическое/православное), Хэллоуин.

### 💱 Валюта

15 валют с возможностью обмена в автономном режиме.

### ⚙️ Настройки

- **Язык**: Английский / Русский (автоматическое определение)
- **Тема**: ☀️ Светлая / 🌙 Темная / 🔄 Авто (системные настройки)
- **Загрузка для Android**: Доступно только для веб-пользователей (скрыто в приложении Tauri)

## 🚀 Быстрый старт

```bash
cd calculator-app
npm install
npm run start
```

## 📦 Сборка для продакшена

```bash
npm run build
```

## 🌐 Развертывание на GitHub Pages

```bash
npm run deploy
```

## 🤖 Сборка Android APK

```bash
bash build-apk.sh
```

Вывод: `apk/SmartCalc.apk` (~32 МБ)

### Необходимые компоненты

- Android SDK с NDK 25.2.9519653
- Java Development Kit (JDK)
- Хранилище ключей по адресу `apk/keystore.jks` (пароль: `smartcalc123`)

## 🎨 Особенности пользовательского интерфейса

- Адаптивный дизайн (mobile-first)
- Плавные анимации (кнопка переключения, переходы)

## 🧮 Логика калькулятора

Обычный режим: +, -, _, :, √, x², π и e вставляются как символы (вычисление по =) и по модулю
(подсчет остатка) Научный режим: Все как в обычном режиме, но также тригонометрические функции в
градусах, % от = вычисление процентов 10ˣ\
Программистский режим: +, -, _, :, побитовые операторы как двоичные (AND, OR, XOR), NOT унарные,
системы единиц (DEC, BIN, OCT и HEX)
